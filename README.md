# ✨ Resume Tailor

A smart resume editing web app that tailors your resume to match job descriptions using AI.

## 🔍 Features
- ✅ Magic Link Login (Supabase)
- 📝 Paste Resume + JD
- 🤖 AI Tailoring Engine (OpenAI)
- 📄 Resume Preview (Before & After)
- 💾 Save History & Download

## 🛠 Built With
- Next.js 14
- Tailwind + ShadCN UI
- MongoDB + Supabase
- OpenAI GPT API
